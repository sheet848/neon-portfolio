This project uses:

- Framework: TanStack Start v1 (full-stack React 19 with SSR, file-based routing in src/routes/)
- Build tool: Vite 7
- Language: TypeScript (strict mode)
- Styling: Tailwind CSS v4 (configured via src/styles.css with native @import and theme variables)
- UI components: shadcn/ui (Radix primitives in src/components/ui/)
- Data/state: TanStack Query + TanStack Router
